## 🌟 功能 Features

### 自訂水平滾動

- 使用「前進 / 後退」按鈕控制水平滾動。
- 根據容器與元素寬度計算滾動距離，處理邊界避免內容被遮擋。
- 提升橫向瀏覽的滑順度與可用性。
  ![Movie rankings](assets/movie-rankings.png)

### 彈出視窗

- 點擊排行榜項目即可開啟 Modal，顯示完整影片資訊。
- 支援背景點擊或按鈕關閉視窗。
  ![Modal](assets/modal.png)

### FAQ 手風琴

- 點擊問題即可展開或收合內容。
- 保證同一時間僅有一個問題展開。
  ![Accordion](assets/accordion.png)

### 小螢幕按鈕顯示/隱藏

- 視窗寬度 ≤ 600px 時啟用滾動事件監聽。
- 根據滾動位置動態顯示或隱藏操作按鈕。
- 使用節流技術避免過於頻繁觸發，提高效能。
